C++ Cheatsheet

About C++

  • C++ is a general-purpose programming language developed by Bjarne Stroustrup in the early 1980s as an extension of the C language. It is often called “C with classes” because it retains the efficiency and low-level features of C while adding powerful features like object-oriented programming (OOP), templates, exception handling, and the Standard Template Library (STL).
  • Being a multi-paradigm language, C++ supports procedural, object-oriented, and generic programming, making it flexible for different types of applications.
  • The language is widely used because it combines the speed and hardware-level control of C with higher-level abstractions that simplify large-scale software development.
  • Through OOP concepts such as encapsulation, inheritance, polymorphism, and abstraction, C++ allows modularity, code reusability, and better organization. It also provides strong memory management using new and delete, supports function and operator overloading, and enables generic programming with templates
  • C++ has a wide range of applications, including system software, operating systems, game development, embedded systems, GUI-based applications, and high-performance computing
  • Its main advantages are efficiency, flexibility, and portability, though it also has challenges such as complexity and the risk of errors in manual memory management.
  • Despite newer languages, C++ remains one of the most important and widely used programming languages in the world due to its balance of performance and abstraction.

1.Basic Program Structure

C++ programs begin with header files like iostream for input and output operations. Every program has a main() function, which is the entry point. Output is displayed using cout, while input is taken using cin. Statements are ended with a semicolon, and code blocks are enclosed in curly braces {}. C++ also supports namespaces, where the standard one (std) contains most built-in functions and objects.

Programs start with main() function.
Uses header files like iostream for I/O.
Namespace (std) avoids naming conflicts.

2. Data Types

C++ supports multiple data types to represent different kinds of values. The common ones are int for integers, float and double for decimal numbers, char for single characters, string for text, and bool for true/false values. Variables must be declared before use, and C++ is a statically typed language, meaning the type of a variable cannot change during execution.

int → stores integers.
float → stores decimal numbers (single precision).
double → stores larger decimal numbers (double precision).
char → stores single characters.
void → means "no value".

3. Variables and Constants

Variables in C++ are named memory locations used to store data that can be modified during program execution. They must be declared with a data type such as int, float, char, or string, which defines the kind of values they hold. Constants, on the other hand, are fixed values that cannot be changed once defined. They can be declared using the const keyword or #define preprocessor directive. Using constants improves code readability and ensures important values remain unchanged throughout the program.

Variable → a storage location with a name, whose value can change.
Constant → fixed value that cannot be modified (declared using const).

4. Input and Output

C++ uses the standard input and output streams provided in the iostream library. The cout object, combined with the insertion operator (<<), is used for displaying output to the console, while the cin object with the extraction operator (>>) is used to take input from the user. For handling multiple inputs in a single line, values are separated by spaces. Additionally, the getline() function is available to read an entire line of text including spaces, which makes it useful for processing strings.

cin → input.
cout → output.
Uses << and >> operators.

5. Operators

Operators are special symbols used to perform operations on data. Arithmetic operators like +, -, *, /, and % handle mathematical calculations. Relational operators such as ==, !=, >, <, >=, and <= are used for comparison. Logical operators &&, ||, and ! are used for decision making. Assignment operators (=, +=, -=, etc.) assign values to variables, while increment and decrement operators (++ and --) increase or decrease values.

Arithmetic → addition, subtraction, multiplication, division, modulus.
Relational → comparisons (greater, smaller, equal, not equal).
Logical → AND, OR, NOT.
Assignment → assigns or updates values.
Increment/Decrement → increases or decreases value by 1.

6.Conditional Statements

Control statements allow the program to make decisions and repeat actions. The if-else statement executes code based on conditions, while the switch statement selects one case from many options. Loops are used for repetition: for is used when the number of iterations is known, while continues until a condition becomes false, and do-while ensures the loop runs at least once before checking the condition.

if → executes when condition is true.
if-else → executes one block if true, another if false.
else if ladder → multiple conditions.

7. Loops

Loops in C++ allow repeating a block of code multiple times until a condition is satisfied. The for loop is commonly used when the number of iterations is known in advance. The while loop executes as long as the condition remains true, making it useful for indeterminate iterations. The do-while loop ensures the code runs at least once before checking the condition, which is helpful in menu-driven programs. Loops are essential for tasks like iterating over arrays, performing calculations, and handling repetitive logic efficiently.

for loop → runs a block for a fixed number of times.
while loop → runs as long as condition is true.
do-while loop → runs at least once, then checks condition.

8.Functions

Functions are blocks of reusable code that perform specific tasks. They can accept parameters and return values. C++ supports both user-defined and built-in functions. Functions promote modularity and reusability in programming. Function overloading, a feature of C++, allows multiple functions with the same name but different parameter lists.

Can return values or be void.
Default arguments → values used if none passed.
Inline functions → faster execution.

9.Arrays

Arrays are collections of elements of the same type stored in contiguous memory. They allow quick access to elements using indexes.Arrays are useful for fixed-size collections, while strings are widely used for text processing.

collections of elements of the same type stored in contiguous memory
useful for fixed-size collections

10.Strings

Strings, although technically character arrays, are handled using the string class in C++, which provides built-in methods like size(), substr(), and concatenation.

provides built-in methods like size(), substr(), and concatenation
provides built-in methods like size(), substr(), and concatenation

11.Memory Management

Memory management in C++ can be done both automatically and manually. Local variables are stored in stack memory, and objects created dynamically are stored in heap memory. The new operator is used to allocate memory at runtime, while the delete operator is used to free that memory to prevent memory leaks. For arrays, new[] and delete[] are used. Proper memory management is important in C++ because it does not have garbage collection like some higher-level languages, so programmers must manage memory explicitly.

new → allocate memory.
delete → free memory.
Replaces C’s malloc and free.

12. Pointers & References

Pointers are variables that store the memory address of other variables. They allow direct manipulation of memory, which is powerful but requires careful handling. Using the & operator, we get the address of a variable, and with the * operator, we access the value stored at that address (dereferencing). Pointers are heavily used in dynamic memory allocation, arrays, and data structures like linked lists.

Pointer → variable that stores address of another variable.
Reference → alias for a variable, must be initialized at declaration.
Supports pointers to objects.

13.Templates

Templates in C++ allow writing generic and reusable code. Instead of creating multiple versions of a function or class for different data types, templates let us define one version that works with any type. Function templates make functions type-independent, while class templates are used to design generic data structures like stacks, queues, or linked lists. This feature is the foundation of the Standard Template Library (STL) and plays a crucial role in making C++ highly flexible and powerful.

Function template → generic functions.
Class template → generic classes.
Allows type-independent programming.

14.Exception Handling

Exception handling in C++ provides a way to manage runtime errors gracefully without crashing the program. It uses the keywords try, catch, and throw. Code that may generate an error is placed inside the try block, and if an exception occurs, it is thrown using the throw keyword. The error is then caught and handled in the catch block. This mechanism is useful for dealing with errors like invalid inputs, division by zero, or file handling failures, ensuring the program runs smoothly under unexpected conditions.

Uses try, catch, throw.
Helps handle runtime errors gracefully.

15.STL (Standard Template Library)

The STL is a collection of generic classes and functions that save time in implementing common data structures and algorithms. It includes containers like vectors (dynamic arrays), sets (unique sorted values), maps (key-value pairs), and sequences like stacks and queues. Iterators are used to traverse elements, and algorithms like sort, find, and reverse make operations easier. STL is widely used in competitive programming for efficiency.

Containers → vector, list, deque, set, map, stack, queue.
Iterators → behave like pointers for traversing containers.
Algorithms → sort, search, reverse, etc.

16..Object-Oriented Programming (OOP)

OOP in C++ introduces classes and objects. A class is a blueprint that defines attributes (data) and methods (functions), while an object is an instance of that class. Features like constructors initialize objects, and destructors clean them up.

Class → blueprint,Object → instance.
Encapsulation → hide data using private/protected/public.
Abstraction → show essential features, hide implementation.
Inheritance → derive classes (single, multiple, multilevel, hybrid).
Polymorphism →(1.Compile-time → function/operator overloading.),(Run-time → virtual functions.)

17.Constructors & Destructors

Constructors and destructors are special member functions of a class in C++. A constructor is automatically invoked when an object is created, and it is used to initialize the object’s attributes. Constructors can be overloaded, meaning a class can have multiple constructors with different parameter lists. A destructor, denoted by a tilde (~) before the class name, is automatically invoked when the object goes out of scope, and it is used to release resources or perform cleanup tasks. Together, they manage the lifecycle of an object effectively.

Constructor → special method auto-called when object is created.
Destructor → special method auto-called when object is destroyed.

18.Files Handling

C++ supports file input and output using the library. Files can be opened in read mode (ifstream), write mode (ofstream), or both (fstream). Writing to files is done using the << operator, while reading uses >> or getline. File handling is useful for storing data permanently, reading configurations, and processing large datasets.

Binary Files:Files can also be read/written in binary format using .read() and .write() functions for efficiency
useful for storing data permanently, reading configurations, and processing large datasets.
